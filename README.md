# TripSorter

requires
* Node.js > 6.5.0
* npm > 3.10.10

## install
```sh
npm install
```

## development
```sh
npm run start
```
app will be running on localhost:8080

## build
```sh
SET NODE_ENV=production
npm run build
```
all assets will be in /dist folder
you can run the app locally by opening index.html

## deploy
```sh
npm run deploy
```
deploys to https://6finger.github.io/pf-travel
